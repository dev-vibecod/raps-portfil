# Project preview assets (Task 2)

All 8 project previews were built in Figma as **representative mockups** (not the live
products). Open each link, **export/screenshot the frame**, and save the PNG into
`public/previews/` using the exact filename below. The site picks them up automatically;
until a file exists, the card shows a labelled placeholder.

> Tip in Figma: select the frame → right panel **Export** → PNG (2x) for a crisp image.

## Notes

- Filenames map to each project's `slug` in `data/content.js` (field `preview`).
- Recommended export ratio for UI cards: ~16:10. The card and modal use `object-cover`,
  so any reasonable size works.
- Every card and the detail modal show a "Representative mockup — not the live product"
  badge, so assets are clearly illustrations.
